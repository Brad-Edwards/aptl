# Archived Legacy Scenario YAML

These files are historical APTL-local scenario fixtures from before the
ADR-035 ACES SDL cutover. They are reference-only evidence for older docs and
parity notes.

They are not catalog entries, not startup inputs, and not parsed by the public
lab runtime. Current scenario authoring and runtime handoff use ACES SDL files
through `scenarios/catalog.json`, `aptl.core.scenario_catalog`, and
`aces_sdl.parse_sdl_file`.
