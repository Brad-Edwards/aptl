Interactive Console in the web UI (`/console`) for exploring APTL: run separate red and blue AI chat sessions with isolated transcripts, control which MCP servers each session can reach, and share findings across sessions via named scratchpads. Works in a no-key demo mode (slash commands run the session's tools for real) and upgrades to a live Claude agent with the new `[console]` extra and an `ANTHROPIC_API_KEY`.
