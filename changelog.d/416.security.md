### Security

Loopback-bound the host-published SOC management surfaces so they are no longer
reachable from other machines on the operator's LAN (ADR-034 Host Exposure
Amendment, issue #416). Cortex (`9001`, which mounts the host `docker.sock`),
TheHive (`9000`), MISP (`8443`), the Shuffle frontend (`3443`/`3001`), and the
Wazuh manager (`1514`/`1515`/`514`/`55000`), indexer (`9200`), and dashboard
(`443`) now publish on `127.0.0.1` only, matching the existing `grafana-otel` /
`web-api` precedent. Container-side ports and the security-net addresses are
unchanged, so in-cluster traffic and host-side clients using `localhost` are
unaffected; only off-host LAN reachability is removed. Deliberately vulnerable
victim targets (e.g. `webapp:8080`) intentionally remain published on all
interfaces. Replacing the raw `docker.sock` mounts with a scoped socket proxy
is tracked as a separate follow-up.
