**Fixed a fresh-checkout lab-start failure: `APTL_API_TOKEN` was undefined.**
`docker-compose.yml` hard-requires `APTL_API_TOKEN` (the ADR-039 web
control-plane bearer token, `${APTL_API_TOKEN:?...}`), but `.env.example` never
defined it, so a `.env` copied from the template failed `aptl lab start`/`stop`
at Compose interpolation. The variable is now in `.env.example` and is included
in the startup placeholder check, so the lab fails closed on the placeholder
token rather than booting an authenticated surface with a guessable value.
