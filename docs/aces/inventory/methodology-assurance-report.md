# ACES Inventory Methodology Assurance Report Has Moved

APTL no longer owns or republishes the asset-inventory methodology assurance
report. The canonical ACES report lives at:

<https://github.com/Brad-Edwards/aces/blob/dev/docs/aces/inventory/methodology-assurance-report.md>

Keep APTL-specific assurance claims in the per-asset evidence bundles and
ledger records. The `aptl aces-inventory` commands remain the current
downstream reference tooling for schema inspection, ledger validation, and gap
reporting, but the methodology and assurance rationale are ACES-owned.
