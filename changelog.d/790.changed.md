`ssh.ts` in `aptl-mcp-common` is split into a shared contracts leaf, the
`PersistentSession` lifecycle module, and the `SSHConnectionManager`
orchestration module, behind a thin `ssh.ts` compatibility facade. Public
exports and runtime behavior are unchanged; the split brings every source
file below the 500-line budget. `PersistentSession`'s constructor and
`SSHConnectionManager.createSession` now take a `SessionConnectOptions`
object (`port`/`mode`/`timeoutMs`/`shellType`) in place of the trailing
positional parameters; defaults are unchanged.
