### Security

Verified TLS for SOC stack clients via a lab-managed CA (SEC-006 / ADR-034).
`aptl lab start` now generates a runtime-only lab CA plus per-service
certificates for MISP, TheHive, Cortex, and Shuffle. The Python sync service,
the run-archive collectors, and the SOC-stack MCP servers (`mcp-threatintel`,
`mcp-casemgmt`, `mcp-soar`) verify TLS against the lab CA by default; the
`MISP_VERIFY_SSL=false` / per-MCP `verify_ssl: false` escape hatches remain
for local debugging only. SEC-004's `rejectUnauthorized: false` allowance is
now narrowed to Wazuh inter-component traffic only.
